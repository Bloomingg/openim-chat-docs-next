群组整体禁言和单个成员禁言是两个状态。群主可管理管理员和普通成员；管理员只能管理普通成员；最终权限由 OpenIMServer 校验。

## 群组禁言

`changeGroupMute:isMute:onSuccess:onFailure:` 接收目标 `groupID` 与开关值。只有有管理权限的成员可以修改，最终由 OpenIMServer 审核。

```objc
[[OIMManager manager] changeGroupMute:groupID
    isMute:YES
    onSuccess:^(NSString * _Nullable data) { [self showGroupMuted]; }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

`isMute` 为 `YES` 时开启，为 `NO` 时关闭。固定 iOS tag 的公开 selector 没有 WASM `muteBypassUserIDs` 参数，不能为此版本文档增加例外用户列表。

## 成员禁言

### 参数说明

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `groupID` | `NSString *` | 目标群组 ID。 |
| `userID` | `NSString *` | 要禁言或解除禁言的成员 ID。 |
| `mutedSeconds` | `NSInteger` | 禁言时长，单位为秒；`0` 表示解除。 |

群主可管理管理员和普通成员，管理员只能管理普通成员；不能仅凭本地角色隐藏来替代服务端权限校验。

```objc
[[OIMManager manager] changeGroupMemberMute:groupID
    userID:targetUserID
    mutedSeconds:3600
    onSuccess:^(NSString * _Nullable data) { [self showMemberMuted]; }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

`mutedSeconds` 单位为秒，传 `0` 解除禁言。`OIMGroupMemberInfo.muteEndTime` 也是秒级截止时间，应以服务端同步值计算状态。

群禁言成功后，其他界面可能通过 `onGroupInfoChanged:` 收到 `OIMGroupInfo`；成员禁言通过 `onGroupMemberInfoChanged:` 收到 `OIMGroupMemberInfo`。成功 callback、事件到达和重新查询是三个阶段。完整 listener 分别见[获取和搜索群组](/sdk/ios/group/retrieving-groups/retrieve-and-search-groups)和[获取群成员](/sdk/ios/group/retrieving-group-members/retrieve-group-members)。
