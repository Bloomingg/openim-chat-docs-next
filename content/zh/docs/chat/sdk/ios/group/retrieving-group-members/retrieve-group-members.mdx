群成员资料由 `OIMGroupMemberInfo` 表示，本地缓存应使用 `groupID:userID` 作为键。

## 分页获取成员

### 参数说明

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `groupID` | `NSString *` | 目标群组 ID。 |
| `filter` | `OIMGroupMemberFilter` | 全部、群主、管理员或普通成员等角色筛选。 |
| `offset` | `NSInteger` | 分页起始位置，第一页为 `0`。 |
| `count` | `NSInteger` | 本次读取数量。 |

```objc
[[OIMManager manager] getGroupMemberList:groupID
    filter:OIMGroupMemberFilterAll
    offset:0
    count:50
    onSuccess:^(NSArray<OIMGroupMemberInfo *> *members) {
        [self replaceFirstPage:members];
    }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

继续加载时按已请求条目数增加 `offset`，不要使用事件合并后的数组长度；返回少于 `count` 时可视为到达当前边界。分页期间收到成员增删事件时，重置为 `offset:0` 重新校准。

固定版没有独立的“被禁言成员列表”方法。需要该视图时分页读取成员，并筛选 `muteEndTime` 晚于当前秒级时间戳的记录。

## 获取指定成员

```objc
[[OIMManager manager] getSpecifiedGroupMembersInfo:groupID
    usersID:userIDs
    onSuccess:^(NSArray<OIMGroupMemberInfo *> *members) {
        [self mergeMembers:members];
    }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

只需要判断哪些用户仍在群内时，使用 `getUsersInGroup:userIDs:onSuccess:onFailure:` 返回命中的 ID。只需要群主和管理员时，使用 `getGroupMemberOwnerAndAdmin:onSuccess:onFailure:`。

## 搜索成员

`keywordList` 在固定版本只应传入一个非空关键词；`isSearchUserID` 和 `isSearchMemberNickname` 至少开启一个。

### 参数说明

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `groupID` | `NSString *` | 目标群组 ID。 |
| `keywordList` | `NSArray *` | 单一非空搜索关键词。 |
| `isSearchUserID` | `BOOL` | 是否匹配用户 ID。 |
| `isSearchMemberNickname` | `BOOL` | 是否匹配群内昵称。 |
| `offset` | `NSInteger` | 搜索分页起点。 |
| `count` | `NSInteger` | 本次返回数量。 |

```objc
OIMSearchGroupMembersParam *param = [OIMSearchGroupMembersParam new];
param.groupID = groupID;
param.keywordList = @[keyword];
param.isSearchUserID = YES;
param.isSearchMemberNickname = YES;
param.offset = 0;
param.count = 50;

[[OIMManager manager] searchGroupMembers:param
    onSuccess:^(NSArray<OIMGroupMemberInfo *> *members) { [self showMatches:members]; }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

## 合并成员事件

查询建立快照；稳定的 listener 对象负责后续增量，并在作用域结束时移除：

```objc
- (void)startMemberEvents { [[OIMManager callbacker] addGroupListener:self]; }
- (void)stopMemberEvents { [[OIMManager callbacker] removeGroupListener:self]; }

- (void)onGroupMemberAdded:(OIMGroupMemberInfo *)memberInfo { [self upsertMember:memberInfo]; }
- (void)onGroupMemberDeleted:(OIMGroupMemberInfo *)memberInfo { [self removeMember:memberInfo]; }
- (void)onGroupMemberInfoChanged:(OIMGroupMemberInfo *)changeInfo { [self upsertMember:changeInfo]; }
```

事件可来自其他管理员或其他客户端，并可能重复到达。分页期间发生新增或删除时，从第一页重新建立快照。
