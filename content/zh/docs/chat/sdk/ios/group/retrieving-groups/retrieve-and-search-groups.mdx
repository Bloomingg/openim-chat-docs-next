## 获取指定群组

`getSpecifiedGroupsInfo:onSuccess:onFailure:` 接收群组 ID 数组并返回 `NSArray<OIMGroupInfo *> *`。读取单个群组时仍传数组，并检查第一项是否存在。

```objc
[[OIMManager manager] getSpecifiedGroupsInfo:@[groupID]
    onSuccess:^(NSArray<OIMGroupInfo *> *groups) {
        OIMGroupInfo *group = groups.firstObject;
        if (group) [self renderGroup:group];
    }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

## 分页获取已加入群组

群组较多时使用 `getJoinedGroupListPageWithOffset:count:onSuccess:onFailure:`；`offset` 从 `0` 开始，继续加载时按已请求条目数递增。

```objc
[[OIMManager manager] getJoinedGroupListPageWithOffset:0 count:50
    onSuccess:^(NSArray<OIMGroupInfo *> *groups) {
        [self replaceFirstPage:groups];
    }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

固定版也有无分页的 `getJoinedGroupListWithOnSuccess:onFailure:`，大型列表应优先分页。`isJoinedGroup:onSuccess:onFailure:` 返回 `BOOL`，用于校准当前用户是否仍在群内。

## 搜索已加入群组

`searchGroups:onSuccess:onFailure:` 只搜索已经同步到本地的已加入群组，不是公开群目录。

```objc
OIMSearchGroupParam *param = [OIMSearchGroupParam new];
param.keywordList = @[keyword];
param.isSearchGroupID = YES;
param.isSearchGroupName = YES;

[[OIMManager manager] searchGroups:param
    onSuccess:^(NSArray<OIMGroupInfo *> *groups) { [self showMatches:groups]; }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

## 保持群组快照同步

由应用级稳定对象实现 `OIMGroupListener`，初始化后添加，作用域结束时移除。查询建立快照，listener 合并后续增量；四类事件都按 `groupID` 幂等处理。

```objc
- (void)startGroupEvents {
    [[OIMManager callbacker] addGroupListener:self];
}

- (void)stopGroupEvents {
    [[OIMManager callbacker] removeGroupListener:self];
}

- (void)onJoinedGroupAdded:(OIMGroupInfo *)groupInfo { [self upsertGroup:groupInfo]; }
- (void)onJoinedGroupDeleted:(OIMGroupInfo *)groupInfo { [self removeGroupID:groupInfo.groupID]; }
- (void)onGroupInfoChanged:(OIMGroupInfo *)changeInfo { [self upsertGroup:changeInfo]; }
- (void)onGroupDismissed:(OIMGroupInfo *)changeInfo { [self markDismissed:changeInfo.groupID]; }
```

事件可能来自其他端并重复到达。分页期间收到新增或删除事件时，重置分页并从第一页重新加载。
