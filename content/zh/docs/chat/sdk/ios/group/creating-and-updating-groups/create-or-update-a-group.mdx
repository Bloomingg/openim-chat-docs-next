## 创建群组

`createGroup:onSuccess:onFailure:` 接收 `OIMGroupCreateInfo`。普通成员、管理员和群主分别配置，三者不要重复。

### 参数说明

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `groupInfo` | `OIMGroupBaseInfo *` | 群名称、头像、公告、介绍、类型和扩展字段。 |
| `memberUserIDs` | `NSArray<NSString *> *` | 初始普通成员 ID。 |
| `adminUserIDs` | `NSArray<NSString *> *` | 初始管理员 ID。 |
| `ownerUserID` | `NSString *` | 群主 ID；使用当前用户时也应显式传入。 |

```objc
OIMGroupBaseInfo *base = [OIMGroupBaseInfo new];
base.groupName = @"项目讨论组";
base.groupType = OIMGroupTypeWorking;
base.faceURL = @"https://example.com/group.png";

OIMGroupCreateInfo *request = [OIMGroupCreateInfo new];
request.groupInfo = base;
request.memberUserIDs = @[@"user-003", @"user-004"];
request.adminUserIDs = @[@"user-002"];
request.ownerUserID = currentUserID;

[[OIMManager manager] createGroup:request
    onSuccess:^(OIMGroupInfo *group) {
        [self openGroupWithID:group.groupID];
    }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

成功 callback 的 `group` 是新建群组的 `OIMGroupInfo`。当前账号的群组列表随后可通过 `onJoinedGroupAdded:` 合并增量。

## 更新群资料

固定版同时提供 `setGroupInfo:onSuccess:onFailure:` 和 `setGroupInfoDictionary:onSuccess:onFailure:`。只更新部分字段时优先使用字典版本，避免用不完整的 `OIMGroupInfo` 覆盖未修改字段。

### 参数说明

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `groupID` | `NSString *` | 必填，目标群组 ID。 |
| `groupName` | `NSString *` | 可选的新群名称。 |
| `notification` | `NSString *` | 可选的群公告。 |
| `needVerification` | `OIMGroupVerificationType` | 入群验证方式。 |
| `applyMemberFriend` | `OIMAllowType` | 是否允许添加群成员为好友。 |
| `lookMemberInfo` | `OIMAllowType` | 是否允许查看群成员资料。 |
| `ex` | `NSString *` | 业务扩展字符串。 |

除 `groupID` 外只放入本次确实要修改的键。群名称、公告、验证方式和成员资料查看权限能否修改，由 OpenIMServer 根据当前用户的群角色审核。

```objc
NSDictionary *changes = @{
    @"groupID": groupID,
    @"groupName": @"新的群名称",
    @"notification": @"本周五发布",
    @"needVerification": @(OIMGroupVerificationTypeAllNeedVerification),
    @"applyMemberFriend": @(OIMAllowTypeAllowed),
    @"lookMemberInfo": @(OIMAllowTypeAllowed),
    @"ex": @"{\"projectID\":\"project-42\"}"
};

[[OIMManager manager] setGroupInfoDictionary:changes
    onSuccess:^(NSString * _Nullable data) { [self showSaved]; }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

`ex` 是完整字符串，SDK 不会合并其中的 JSON 字段。成功 callback 只表示设置请求完成；最终群资料以 `onGroupInfoChanged:` 或重新调用 `getSpecifiedGroupsInfo:onSuccess:onFailure:` 的结果校准。完整 listener 见[获取和搜索群组](/sdk/ios/group/retrieving-groups/retrieve-and-search-groups)。
