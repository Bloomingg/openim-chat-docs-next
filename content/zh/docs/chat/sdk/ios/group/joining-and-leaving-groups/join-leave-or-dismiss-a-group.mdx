## 加入群组

`joinGroup:reqMsg:joinSource:ex:onSuccess:onFailure:` 中，`joinSource` 记录加入途径，不决定是否需要审核。

### 参数说明

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `groupID` | `NSString *` | 要申请加入的群组 ID。 |
| `reqMsg` | `NSString *` | 申请说明；无内容时传空字符串。 |
| `joinSource` | `OIMJoinType` | 邀请、搜索或二维码等加入来源。 |
| `ex` | `NSString *` | 可选业务扩展；没有时传 `nil`。 |

```objc
[[OIMManager manager] joinGroup:groupID
    reqMsg:@"申请加入项目讨论组"
    joinSource:OIMJoinTypeSearch
    ex:nil
    onSuccess:^(NSString * _Nullable data) { [self showApplicationSubmitted]; }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

`OIMJoinType` 可用 `OIMJoinTypeInvited`、`OIMJoinTypeSearch` 或 `OIMJoinTypeQRCode`。需要审核时，成功 callback 只表示申请已提交；只有 `onJoinedGroupAdded:` 到达或重新查询确认后，才能把当前用户视为群成员。

## 退出群组

`quitGroup:onSuccess:onFailure:` 只接收目标 `groupID`。普通成员和管理员可以请求退出；群主必须先转让群主身份，最终权限由 OpenIMServer 校验。

```objc
[[OIMManager manager] quitGroup:groupID
    onSuccess:^(NSString * _Nullable data) { [self closeGroupPage]; }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

退出后的群组增量通过 `onJoinedGroupDeleted:` 按 `groupID` 移除。成功 callback、事件到达和重新查询成员身份是三个阶段。

## 解散群组

`dismissGroup:onSuccess:onFailure:` 只接收目标 `groupID`，并由 OpenIMServer 校验当前用户是否为群主。

```objc
[[OIMManager manager] dismissGroup:groupID
    onSuccess:^(NSString * _Nullable data) { [self showDismissed]; }
    onFailure:^(NSInteger code, NSString *message) {
        [self showErrorCode:code message:message];
    }];
```

只有群主可解散群组。成功 callback 表示请求完成，`onGroupDismissed:` 用于同步其他界面和其他端的状态。解散不同于删除或隐藏会话。完整 listener 归[获取和搜索群组](/sdk/ios/group/retrieving-groups/retrieve-and-search-groups)。
