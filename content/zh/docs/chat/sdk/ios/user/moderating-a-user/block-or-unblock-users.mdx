黑名单是当前用户主动维护的单向关系。群管理员禁言、移除成员和平台封禁属于其他能力。

## 拉黑用户

```objc
[[OIMManager manager] addToBlackList:@"user_b"
                                  ex:@"{\"source\":\"profile-menu\"}"
                           onSuccess:^(NSString *data) {
                               [blacklistStore markRequestCompletedForUserID:@"user_b"];
                           }
                           onFailure:^(NSInteger code, NSString *message) {
                               NSLog(@"addToBlackList failed: %ld %@", (long)code, message);
                           }];
```

取消拉黑：

```objc
[[OIMManager manager] removeFromBlackList:@"user_b"
                                onSuccess:nil
                                onFailure:nil];
```

成功 callback 表示请求完成；完整列表仍通过 delegate 增量或重新查询校准。

## 监听黑名单事件

```objc
@interface BlacklistSync : NSObject <OIMFriendshipListener>
@end

@implementation BlacklistSync
- (void)onBlackAdded:(OIMBlackInfo *)info { [blacklistStore mergeItem:info]; }
- (void)onBlackDeleted:(OIMBlackInfo *)info { [blacklistStore removeUserID:info.userID]; }
@end

[[OIMManager callbacker] addFriendListener:blacklistSync];
// 账号作用域结束时：
[[OIMManager callbacker] removeFriendListener:blacklistSync];
```

列表按 `userID` 合并。需要完整校准时调用 `getBlackListWithOnSuccess:onFailure:`。
