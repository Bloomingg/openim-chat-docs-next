## 更新好友关系资料

```objc
OIMUpdateFriendsReq *req = [OIMUpdateFriendsReq new];
req.friendUserIDs = @[@"user_b"];
req.remark = @"Alex";
req.isPinned = YES;
req.ex = @"{\"source\":\"contacts\"}";

[[OIMManager manager] updateFriends:req onSuccess:nil onFailure:nil];
```

一次调用会把相同字段应用到全部 `friendUserIDs`。`ex` 按完整字符串覆盖；不同好友需要不同备注时应分别调用。

## 删除好友关系

```objc
[[OIMManager manager] deleteFriend:@"user_b"
                         onSuccess:nil
                         onFailure:nil];
```

成功表示请求完成。事件到达和重新查询校准是后续阶段。

## 监听好友事件

```objc
@interface FriendSync : NSObject <OIMFriendshipListener>
@end

@implementation FriendSync
- (void)onFriendAdded:(OIMFriendInfo *)info { [friendStore mergeFriend:info]; }
- (void)onFriendInfoChanged:(OIMFriendInfo *)info { [friendStore mergeFriend:info]; }
- (void)onFriendDeleted:(OIMFriendInfo *)info { [friendStore removeUserID:info.userID]; }
@end

[[OIMManager callbacker] addFriendListener:friendSync];
// 账号作用域结束时：
[[OIMManager callbacker] removeFriendListener:friendSync];
```

所有好友记录按 `userID` 合并。事件未到达时可调用 `getSpecifiedFriendsInfo:filterBlack:...` 或 `checkFriend:...` 校准。
