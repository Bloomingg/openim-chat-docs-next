发送消息分为创建 `OIMMessageInfo` 和提交发送两个步骤。创建方法同步返回本地对象；成功回调返回服务端确认后的消息对象。

## 发送单聊文本消息

```objc
OIMMessageInfo *message = [OIMMessageInfo createTextMessage:@"你好"];

[[OIMManager manager] sendMessage:message
                           recvID:@"user_b"
                          groupID:nil
                     isOnlineOnly:NO
                  offlinePushInfo:nil
                        onSuccess:^(OIMMessageInfo * _Nullable sentMessage) {
    if (sentMessage == nil) {
        [self showSendError:0 message:@"发送成功回调未返回消息对象"];
        return;
    }
    [self upsertMessage:sentMessage];
}
                       onProgress:^(NSInteger progress) {
    [self updateProgress:progress clientMsgID:message.clientMsgID];
}
                        onFailure:^(NSInteger code, NSString * _Nullable errorMessage) {
    [self showSendError:code message:errorMessage ?: @"发送失败"];
}];
```

群聊发送时改为 `recvID:nil`、`groupID:groupID`。不要同时填写两个目标，也不要复用已经成功发送消息的 `clientMsgID`。

### 参数说明

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `message` | `OIMMessageInfo *` | 由消息创建方法生成的待发送消息。 |
| `recvID` | `NSString *` | 单聊目标用户 ID；群聊传 `nil`。 |
| `groupID` | `NSString *` | 群组 ID；单聊传 `nil`。 |
| `isOnlineOnly` | `BOOL` | 为 `YES` 时仅在线投递，不进入普通离线消息流程。 |
| `offlinePushInfo` | `OIMOfflinePushInfo *` | 可选的离线推送标题、描述与扩展信息。 |

`onProgress` 主要用于媒体上传进度。文本消息也应提供稳定的成功与失败处理，不要把调用开始视为发送成功。

## 返回结果与接收

`onSuccess` 的消息参数在固定 callback 声明中可为空；非空时才是服务端确认后的最终 `OIMMessageInfo`。发送端按返回对象的 `clientMsgID` 更新消息状态；其他客户端通过 `onRecvNewMessage:` 收到单条消息。成功回调不代表所有接收端已经执行事件处理。

如果媒体已经由业务自行上传，可使用 `sendMessageNotOss:recvID:groupID:isOnlineOnly:offlinePushInfo:onSuccess:onProgress:onFailure:`，并确保消息元素包含可访问的远端 URL。一般媒体消息应继续使用 `sendMessage:`，由 SDK 管理上传。

## Swift 调用

```swift
let message = OIMMessageInfo.createTextMessage("你好")
OIMManager.manager().sendMessage(
    message,
    recvID: "user_b",
    groupID: nil,
    isOnlineOnly: false,
    offlinePushInfo: nil,
    onSuccess: { sentMessage in
        guard let sentMessage else {
            showSendError(0, message: "发送成功回调未返回消息对象")
            return
        }
        upsertMessage(sentMessage)
    },
    onProgress: { progress in
        updateProgress(progress, clientMsgID: message.clientMsgID)
    },
    onFailure: { code, message in
        showSendError(code, message: message ?? "发送失败")
    }
)
```

完整的新消息监听与清理见[接收消息](/sdk/ios/message/receiving-messages/receive-messages)。
