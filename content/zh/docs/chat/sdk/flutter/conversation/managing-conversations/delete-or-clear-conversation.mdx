固定 Flutter SDK 公开两种处理指定会话及其消息的方法：

| 方法 | 用途 |
| --- | --- |
| `deleteConversationAndDeleteAllMsg()` | 删除指定会话记录及该会话下的消息。 |
| `clearConversationAndDeleteAllMsg()` | 保留会话入口，清空该会话下的消息和摘要状态。 |

固定基线没有公开只删除会话记录的 `deleteConversation()`。`deleteAllConversationFromLocal()` 已废弃，公开文档不使用该方法；需要隐藏全部本地会话入口时使用 `hideAllConversations()`，但这不等同于删除消息。

## 删除会话及消息

```dart
await OpenIM.iMManager.conversationManager
    .deleteConversationAndDeleteAllMsg(conversationID: conversationID);
```

该操作不等于删除好友关系、退出群组或解散群组。聊天目标仍存在时，后续新消息或同步可以重新生成会话。

## 保留会话并清空消息

```dart
await OpenIM.iMManager.conversationManager
    .clearConversationAndDeleteAllMsg(conversationID: conversationID);
```

它适合“清空聊天记录”：会话入口继续保留，但当前账号下该会话的消息和摘要状态会被清理。

## 调用后的状态变化

Future 成功后立即清理当前消息列表，再通过 `onConversationChanged` 或重新查询校准会话摘要、最后一条消息和未读状态。回调携带 `List<ConversationInfo>`，应按 `conversationID` 幂等合并。完整 listener 见[获取会话列表](/sdk/flutter/conversation/retrieving-conversations/retrieve-conversation-list)。
