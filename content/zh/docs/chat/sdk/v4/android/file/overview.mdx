文件上传能力来自 `open_im_sdk/third.go`，用于将本地文件上传到 SDK 配置的对象存储链路。发送文件、图片、音视频消息前，可先通过上传接口获得资源 URL。

## 方法索引

| 方法                                                         | 说明                                                                            |
| ------------------------------------------------------------ | ------------------------------------------------------------------------------- |
| [上传文件](/docs/chat/sdk/v4/android/file/methods/upload-file) | 上传文件到 SDK 配置的对象存储链路，适合发送文件、图片、音视频前先完成资源上传。 |

## 上传回调

| 回调                                                                                                                                              | 说明                   |
| ------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------- |
| [`UploadFileCallback`](/docs/chat/sdk/v4/android/models/upload-file-callback).Open(size int64)                                                    | 文件打开并得到总大小。 |
| [`UploadFileCallback`](/docs/chat/sdk/v4/android/models/upload-file-callback).PartSize(partSize int64, num int)                                   | 分片大小和分片数量。   |
| [`UploadFileCallback`](/docs/chat/sdk/v4/android/models/upload-file-callback).HashPartProgress(index int, size int64, partHash string)            | 分片哈希进度。         |
| [`UploadFileCallback`](/docs/chat/sdk/v4/android/models/upload-file-callback).HashPartComplete(partsHash string, fileHash string)                 | 所有分片哈希完成。     |
| [`UploadFileCallback`](/docs/chat/sdk/v4/android/models/upload-file-callback).UploadID(uploadID string)                                           | 获取上传 ID。          |
| [`UploadFileCallback`](/docs/chat/sdk/v4/android/models/upload-file-callback).UploadPartComplete(index int, partSize int64, partHash string)      | 单个分片上传完成。     |
| [`UploadFileCallback`](/docs/chat/sdk/v4/android/models/upload-file-callback).UploadComplete(fileSize int64, streamSize int64, storageSize int64) | 文件上传完成阶段信息。 |
| [`UploadFileCallback`](/docs/chat/sdk/v4/android/models/upload-file-callback).Complete(size int64, url string, typ int)                           | 文件完成并返回 URL。   |

## 相关类型

- [底层文件上传回调](/docs/chat/sdk/v4/android/models/upload-file-callback)（`UploadFileCallback`）：core binding 文件上传回调，Android 调用层通常映射为 OnPutFileListener。
- [文件上传回调](/docs/chat/sdk/v4/android/models/on-put-file-listener)（`OnPutFileListener`）：上传文件时使用的阶段和进度回调。
- [上传参数](/docs/chat/sdk/v4/android/models/put-args)（`PutArgs`）：文件上传参数。
