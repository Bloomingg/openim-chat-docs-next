好友方法来自 `open_im_sdk/relation.go` 中的好友关系部分。它们负责好友资料读取、好友列表分页、好友搜索、申请处理和好友关系更新。

## 方法列表

### GetSpecifiedFriendsInfo

| 项目       | 说明                                                                                              |
| ---------- | ------------------------------------------------------------------------------------------------- |
| 方法签名   | `GetSpecifiedFriendsInfo(callback Base, operationID string, userIDList string, filterBlack bool)` |
| 作用       | 批量获取指定好友的关系资料，适合好友详情页、会话资料补齐或局部刷新好友信息。                      |
| 关键参数   | `userIDList` 是用户 ID 列表 JSON；`filterBlack` 为 `true` 时过滤黑名单用户。                      |
| 结果或事件 | 成功返回好友资料列表 JSON。                                                                       |

### GetFriendList

| 项目       | 说明                                                                 |
| ---------- | -------------------------------------------------------------------- |
| 方法签名   | `GetFriendList(callback Base, operationID string, filterBlack bool)` |
| 作用       | 获取当前用户全部好友列表，适合好友通讯录首屏加载或登录后全量同步。   |
| 关键参数   | `filterBlack` 控制是否排除已拉黑用户。                               |
| 结果或事件 | 成功返回好友列表 JSON。                                              |

| 方法签名   | `GetFriendListPage(callback Base, operationID string, offset int32, count int32, filterBlack bool)` |

| 关键参数   | `offset` 和 `count` 控制分页；`filterBlack` 控制是否排除黑名单用户。                                |
| 结果或事件 | 成功返回当前页好友列表 JSON。                                                                       |

## 回调与返回约定

Android core binding 的异步方法通过回调返回结果。

| 回调类型          | 方法                                    | 说明                                                   |
| ----------------- | --------------------------------------- | ------------------------------------------------------ |
| `Base`            | `OnSuccess(data string)`                | 成功时返回 JSON 字符串、空字符串或基础类型序列化结果。 |
| `Base`            | `OnError(errCode int32, errMsg string)` | 失败时返回 SDK 错误码和错误消息。                      |
| `SendMsgCallBack` | `OnProgress(progress int)`              | 仅发送消息时使用，表示发送或上传进度。                 |

除同步创建消息类方法外，带 `callback` 的方法不直接返回业务数据。参数中名称包含 `List`、`Info`、`Param`、`Req`、`Options` 的 `string` 通常是 JSON 字符串，SDK 内部会反序列化为对应结构。

## 事件联动

好友申请、新增好友、删除好友和好友资料变化通过 `SetFriendListener` 注册的 `OnFriendshipListener` 接收。
