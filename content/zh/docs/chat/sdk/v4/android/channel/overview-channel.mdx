OpenIM Android core 使用群组承载多人聊天能力。旧版结构中的 channel 页面在这里统一映射为 OpenIM 群组；当前 core 没有 open channel 的独立导出方法。

## 方法列表

### CreateGroup

| 项目       | 说明                                                                   |
| ---------- | ---------------------------------------------------------------------- |
| 方法签名   | `CreateGroup(callback Base, operationID string, groupReqInfo string)`  |
| 作用       | 创建一个 OpenIM 群组。适合发起群聊、创建项目群或从业务组织关系生成群。 |
| 关键参数   | `groupReqInfo` 是群创建请求 JSON，包含群资料、群主和成员等字段。       |
| 结果或事件 | 成功返回群资料 JSON，并触发相关群组事件。                              |

### JoinGroup

| 项目       | 说明                                                                                                       |
| ---------- | ---------------------------------------------------------------------------------------------------------- |
| 方法签名   | `JoinGroup(callback Base, operationID string, groupID string, reqMsg string, joinSource int32, ex string)` |
| 作用       | 申请加入指定群组。是否直接入群取决于群设置和服务端审核策略。                                               |
| 关键参数   | `groupID` 是目标群 ID；`reqMsg` 是申请说明；`joinSource` 表示入群来源；`ex` 为扩展信息。                   |
| 结果或事件 | 成功返回申请或入群结果，群相关监听会收到变化。                                                             |

## 回调与返回约定

Android core binding 的异步方法通过回调返回结果。

| 回调类型          | 方法                                    | 说明                                                   |
| ----------------- | --------------------------------------- | ------------------------------------------------------ |
| `Base`            | `OnSuccess(data string)`                | 成功时返回 JSON 字符串、空字符串或基础类型序列化结果。 |
| `Base`            | `OnError(errCode int32, errMsg string)` | 失败时返回 SDK 错误码和错误消息。                      |
| `SendMsgCallBack` | `OnProgress(progress int)`              | 仅发送消息时使用，表示发送或上传进度。                 |

除同步创建消息类方法外，带 `callback` 的方法不直接返回业务数据。参数中名称包含 `List`、`Info`、`Param`、`Req`、`Options` 的 `string` 通常是 JSON 字符串，SDK 内部会反序列化为对应结构。

## 事件联动

群组相关变化通过 `SetGroupListener` 注册的 `OnGroupListener` 接收，包括群新增/删除、群资料变化、成员新增/删除、成员资料变化、群申请新增/删除/接受/拒绝等。
